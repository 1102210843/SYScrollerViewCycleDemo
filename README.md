# SYScrollerViewCycleDemo
循环轮播控件1.0版，使用简单,需要配合SDWebImage使用，支持NSURL，UIImage，NSString等格式及本地图片名的形式

可设置轮播图展示的高度
/**
 *  设置图片自动播放时间，默认2秒
 */                                                                                             
@property (nonatomic, assign) NSTimeInterval timeInterval;  可自定义播放图片切换时间
